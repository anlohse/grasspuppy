# Beginner Curriculum

The Grasspuppy beginner track consists of 20 progressive lessons designed to take a learner from absolute zero to understanding basic programming concepts like functions, variables, logic, and arrays.

## 🎯 Learning Objectives

By the end of this track, the learner should understand:
- Function calls and arguments.
- Basic data types: Strings and Numbers.
- Command sequencing (top-to-bottom execution).
- Visual output and pattern generation.
- Logic branching (`if/else`).
- Repetition and loops (`for`).
- User interaction (`readLine`).
- Collections (Arrays) and randomness.

---

## 📦 Module Structure

### Module 1: First Steps
Focus: Editor navigation and basic output.
1. **Welcome to Grasspuppy** - Meet the tool.
2. **Print your first message** - The `print` function.
3. **Draw your first box** - First visual output.
4. **Colors as values** - Passing arguments to functions.

### Module 2: Sequences and Patterns
Focus: Ordering and layout.
5. **Commands happen in order** - Sequencing.
6. **Draw a row of boxes** - Manual repetition.
7. **Start a new line** - Using `newLine()`.
8. **Create a simple flag** - Basic visual composition.

### Module 3: Variables, Strings, and Input
Focus: Data storage and interaction.
9. **Store a message in a variable** - Introduction to `let`.
10. **Combine strings** - Concatenation.
11. **Ask a question** - Interactive input with `readLine`.
12. **Numbers and Expressions** - Simple arithmetic.

### Module 4: Decisions and Loops
Focus: Control flow.
13. **Make a decision** - Conditional logic with `if`.
14. **Use else** - Alternative execution paths.
15. **Repeat with a for loop** - Automated repetition.
16. **Build patterns with loops** - Grid generation.

### Module 5: Functions, Arrays, and Mini Projects
Focus: Abstraction and creative application.
17. **Create your own function** - Reusable logic.
18. **Use function parameters** - Flexible functions.
19. **Pick from an array** - Lists and `pickRandom`.
20. **Final Project** - Interactive pattern generator.

## 🌐 Localization

All curriculum content is localized via `src/i18n.ts`. Lesson definitions in `src/lessons.ts` use translation keys to ensure content works for both English (US) and Portuguese (BR).

For more details on writing new lessons, see the [Lesson Schema](lesson-schema.md).
